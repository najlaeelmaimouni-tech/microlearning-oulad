# microlearning-oulad
Decision-tree sequencing engine for microlearning, validated on the OULAD dataset
## Data

Derived from the Open University Learning Analytics Dataset (OULAD), licensed CC BY 4.0.

Kuzilek, J., Hlosta, M., & Zdrahal, Z. (2017). Open University Learning Analytics dataset. Scientific Data, 4, 170171.

Original dataset: https://analyse.kmi.open.ac.uk/open_dataset

## Files

- `oulad_pipeline.py` — builds the feature space and trains the decision tree
- `oulad_dataset.csv` — derived dataset (304,597 learner-object pairs)
